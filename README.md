# RacingCar-compose
A racing car game implementation using Jetpack Compose

## Milestones
- [x] Blocker unique positioning
- [x] Movement via accelerometer
- [x] Collision detection
- [x] Highscore
- [x] Sounds
- [ ] Speedometer
- [ ] Dynamic game setting
- [ ] power-ups (e.g. booster, shield)
- [ ] Achievements
- [ ] Online leaderboard
